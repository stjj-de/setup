# Server Setup

- Clone this repository
```bash
git clone https://github.com/stjj-de/setup.git website
cd website
```

- Create `keystone.env`
```bash
cd config

cp keystone.env_example keystone.env
nano keystone.env
```

- [Install docker](https://docs.docker.com/engine/install/#server)

## Update
```bash
./update.sh
```
